Seletor por tipo

Seletor por tipo: Este seletor é aplicado a TODAS as tags do mesmo tipo.

Seletor descendente que só é valido para o conteúdo que está dentro de 'p' + 'span' Pode selecionar um ou mais elementos que estão dentro de outro, ou seja,
que SÂO descendente do elemento principal.

O conteúdo que está dentro de 'span' NÃO recebe a cor porque está fora de 'p'.

Pseudo-classes: São tipos de classes especiais que não são definidas pelo desenvolvedor
porque já são pré-definidas. Exemplo: 'a:hover'

Id

Id(#): Identificação única. Só pode ser utilizada uma vez no documento inteiro.
Normalmente é utilizada para identificar elementos estruturais da página. Exemplo: 'div id=cabecalho'

class

Classes(.): Identificação multipla. Pode ser utilizada multiplas vez no documento inteiro.
Exemplo: 'div class=titulo'
Também pode combinar mais de 2 seletores em um mesmo atributo class. Exemplo: 'div class=cont_destque cont_mais_acessado '
